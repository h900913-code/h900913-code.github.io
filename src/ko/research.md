# 연구 관심사

## 환경 커뮤니케이션

기후 위기와 환경 이슈가 미디어와 공론장에서 어떻게 다루어지고 소통되는지를 연구합니다. 인터넷 기사와 댓글을 분석하여 기후 위기에 관한 담론 구조를 파악하고, 미디어가 환경 인식에 미치는 영향을 탐구합니다.

## 텍스트 마이닝

토픽 모델링, 네트워크 분석 등 컴퓨터 기반 텍스트 분석 방법론을 활용하여 환경·에너지 분야의 대규모 텍스트 데이터에서 의미 있는 패턴과 주제를 추출합니다. 학술 논문, 뉴스 기사, 정책 문서 등 다양한 텍스트 소스를 분석 대상으로 삼습니다.

## 관련 연구 주제

- 기후 위기 담론 분석
- 에너지 정책 및 사회적 수용성
- 남북 기후변화·재생에너지 연구 협력
- IPCC 평가보고서 분석

## 연구 방법론

- 토픽 모델링 (Topic Modeling)
- 네트워크 분석 (Network Analysis)
- 내용 분석 (Content Analysis)
- R, Python, SQL 기반 텍스트 마이닝
- Netminer를 활용한 그래프 데이터 분석
